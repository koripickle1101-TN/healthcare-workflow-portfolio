How to Enable GitHub Pages

  1. Go to the healthcare workflow portfolio repository.
  2. Open Settings.
  3. Go to Pages.
  4. Under Build and deployment, choose Deploy from a branch.
  5. Choose the main branch.
  6. Choose the docs folder.
  7. Save the settings.

LinkedIn Featured Instructions

  1. Go to your LinkedIn profile.
  2. Scroll to Featured.
  3. Choose Add a link.
  4. Paste the live GitHub Pages link.
  5. Use the project title below.
  6. Use the project description below.
  7. Save the featured item.

I built a new healthcare operations portfolio asset.

AI Assisted Revenue Cycle Workflow System

This project maps where revenue cycle breakdowns often begin before a claim reaches billing.

Patient intake
Insurance verification
Prior authorization
Claim submission
Denial follow up

The goal was not to present AI as a replacement for healthcare staff.

The goal was to show how AI can support workflow visibility, denial pattern recognition, reporting, and administrative organization while keeping human review responsible for compliance, documentation accuracy, payer rules, and patient context.

This project connects directly to my focus in healthcare administration, revenue cycle operations, denial prevention, and patient access workflows.
